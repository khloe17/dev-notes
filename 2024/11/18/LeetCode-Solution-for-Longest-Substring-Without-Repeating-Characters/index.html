Question

Given a string s, find the length of the longest substring without repeating characters.

Explanation

返回不重复的最长字符字串。

Idea: Sliding window

  1. Use two pointers to tract the string, left and right represent the start and end of the window (both start from 0)
  2. If there is no duplicated character, add the character to the set. If duplicated character exists, remove the character from left side one by one until there is no duplicated one. Update the max length.
