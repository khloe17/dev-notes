Question

Given two strings s and t of lengths m and n respectively, return the minimum window substring of s such that every character in t (including duplicates) is included in the window. If there is no such substring, return the empty string “”.

Explanation

返回包含所有给定字母的最小字串长度，没有则返回””.

Idea: Sliding window

  1. Setup the frequency map for t: create a frequency map to record the frequency and number of unique characters (what we need).
  2. Initialize sliding window variables: use two pointers to define the window. use a variable have to count how many unique characters from t have met their required frequency (what we have). use ans to store the window info (length, start, end).
  3. Expand the window with right pointer: move until end of s, adding characters to the window and update the window info and count about what we have.
  4. Shrink the window with left: when have == need, we start to contract the window to minimize its size. After moving the left pointer, we update window info if needed.
  5. After contracting the window, we continue to move right pointer until the end of the string.
