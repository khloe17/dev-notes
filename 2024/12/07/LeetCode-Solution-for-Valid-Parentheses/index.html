Question

Given a string s containing just the characters ‘(‘, ‘)’, ‘{‘, ‘}’, ‘[‘ and ‘]’, determine if the input string is valid.

An input string is valid if:

Open brackets must be closed by the same type of brackets.
Open brackets must be closed in the correct order.
Every close bracket has a corresponding open bracket of the same type.

Explanation

查看符号开关是否一一对应

Input: s = “()[]{}”
Output: true

Input: s = “(]”
Output: false

Input: s = “([{}])”
Output: true

Idea: Stack (Last In, First Out)

  1. When encountering an open bracket ((, {, [), push it onto the stack
  2. When encountering a close bracket (), }, ]), check the top of the stack. If matches, pop it from the stack. Otherwise, the string is valid.
  3. After processing the entire string, the stack should be empty.

Conditions for validity: 1. match pair 2. if the stack is empty
