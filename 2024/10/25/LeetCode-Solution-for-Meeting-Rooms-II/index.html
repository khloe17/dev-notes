Question

Given an array of meeting time intervals consisting of start and end times [[s1,e1],[s2,e2],…] (si < ei), find the minimum number of conference rooms required.

Explanation

一个数组包含多个会议区间，每个区间都有其开始时间和结束时间，找出所需满足所有开会区间的最少会议室数量。

Idea : Sorting + Two Pointers

  1. 将各区间的开始时间和结束时间分别排序，并用两个指针分别遍历这两个数组
  2. 比较判断：
    遍历条件为：开始数组对应指针仍在数组长度范围内 (for start in start_times)。遍历完所有开始时间，表明所有会议都完成。
    如果开始时间指针对应数字 >= 结束时间时间指针数字对应数组，说明会议无重叠，结束时间数组指针右移到下一位；否则出现会议重叠，更新会议室数量 (rooms += 1)
