LeetCode Solution for Set Matrix Zeroes

Created2024-11-09|Updated2024-11-11|LeetCodeAlgorithm
|Post Views:

Question

Given an m x n integer matrix matrix, if an element is 0, set its entire row and column to 0’s.

You must do it in place.

Explanation

如果矩阵中某个格子的值是 0，则该格子对应的行和列全部设置为 0，返回变更后的矩阵。

eg. input:
matrix = [
[1, 1, 1],
[1, 0, 1],
[1, 1, 1]
]

output:
matrix = [
[1, 0, 1],
[0, 0, 0],
[1, 0, 1]
]

Idea

  1. Initial matrix check (first row and first column): check if first row and first column has zero. If has, set the value to True; otherwise False. (e.g. first_row_has_zero = True, first_col_has_zero = False)
  2. Mark row and columns: Traverse from the second row and second column (Separate Flags for First Row/Column). If we find matrix[i][j] = 0, then we set matrix[i][0] = 0, and matrix[0][j] = 0 (marker for changing to 0 later)
  3. Set cells based on markers: Traverse the matrix again (excluding the first row and first column). Based on the 0 marker, we set the corresponding cells into zero.
  4. Zero out the first row and column. (因为第一行和第一列我们之前是作为标记行列来操作，如果之前这第一行和列包含 0，即 value = True，我们此时再把对应的第一行或者列全部更改为 0。如果不包含，此时我们不需要再进行操作。)
